# 记录

## 一些常用的命令

- `:nmap` 查询映射
- `:lua print(vim.inspect(xxx))` 打印一个 table
  - `print(tbl)` 打印的是一个 table 的内存地址
  - `print(vim.inspect(tbl))` 打印的一个 table 内容的字符串
- `:Telescope help_tags` 模糊查询命令
- `:source %` 重新执行当前 buffer 的代码
- `:e xx/xx` 编辑文件
- `:! ls -al` 运行命令

- `:left` 将当前行文本左对齐
- `:center` 将当前行文本居中对齐
- `:right` 将当前行文本右对齐
